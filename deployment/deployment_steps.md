#  ERC-20 Token Deployment Steps

## Tools Required
- Remix IDE (https://remix.ethereum.org)
- MetaMask Wallet
- Sepolia Test Network


##  Steps to Deploy

1. Open Remix IDE  
2. Create file: MyToken.sol  
3. Paste smart contract code  
4. Go to Solidity Compiler  
   - Version: 0.8.20+  
5. Compile contract  
6. Open Deploy & Run tab  
7. Connect MetaMask wallet  
8. Select Sepolia Test Network  
9. Deploy contract with initial supply (e.g., 1000)  
10. Confirm transaction in MetaMask  



## Post Deployment

- Copy contract address  
- View on Sepolia Etherscan  
- Test token transfer functions
