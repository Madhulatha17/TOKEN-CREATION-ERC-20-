#  Project Overview  

This project demonstrates the creation and deployment of a **custom ERC-20 token** using Solidity on the Ethereum blockchain.

The token is built using the **OpenZeppelin ERC-20 standard**, ensuring security, reliability, and compatibility with Ethereum wallets and decentralized applications.

# Objectives  

- Create a custom ERC-20 token using Solidity  
- Understand token standards in blockchain  
- Deploy smart contract on Ethereum testnet  
- Learn token minting and transfer mechanism  
- Gain hands-on blockchain development experience
